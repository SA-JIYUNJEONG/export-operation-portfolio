# export-operation-portfolio
T## 포트폴리오 개요
본 포트폴리오는 수출 포워딩 업무의 실무 흐름과 수출 서류 관리 구조를 이해하기 위해 제작한 데모 프로젝트입니다.

포워딩 실무에서 사용되는 수출 업무 프로세스와 문서 흐름을 정리하고,
이를 기반으로 간단한 업무 관리 데모 화면을 구현하였습니다.

## 구성 내용
- 수출 포워딩 업무 흐름 정리
- 수출 서류 관리 구조
- 포워딩 운영 프로세스 이해
- 수출 업무 관리 데모 화면

## 데모 실행 방법
저장소에 있는  HTML 파일 이름 클릭 후 다운로드하여 브라우저에서 실행하면 데모 화면을 확인할 수 있습니다.
